Rules of the game:

Guess the word one letter at a time. Only letters are allowed.
Each wrong guess draws part of the hangman.
You lose after 6 wrong guesses.
Press the 'Help' button to view the rules in-game.
Press the 'Reset Game' button to rerset the game state.

Instructions to install:

Ensure node is installed and that you are in the project directory.
Run 'npm install' to install dependencies.
Run 'npm start' to start the app on localhost:3000.
